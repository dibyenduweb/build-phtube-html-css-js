Discuss the scope of var, let, and const.?

var:-var is a keyword.used for declare a variable in javascript.

let:-let is a keyword.used for declare a block scoped variable in javascript.

const:-const is a keyword.used for declare a variable with a constant value in javascript.


What do you mean by REST API.?

A REST API (Representational State Transfer Application Programming Interface) A REST API in JavaScript is a standardized way to communicate with web services over the internet, using HTTP methods like GET, POST, PUT, and DELETE to access and manipulate data or perform actions on remote servers, typically exchanging data in JSON format.


Tell us the use cases of null and undefined

null:Typically represents the absence of an expected value, such as uninitialized variables, missing function arguments, or accessing nonexistent object properties

undefined:Explicitly indicates that a variable or object property has no value or is empty, and it can serve as a placeholder for expected data that hasn't been provided yet.
